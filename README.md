# Тёплые слова ✨

Маленький проект для тёплых слов коллегам — сайт и Telegram-бот, которые раздают персональные комплименты.

## 🌸 Что внутри

- **`index.html`** — статичный сайт с двумя экранами: выбор имени → персональный комплимент. Плавающие эмодзи, розовый градиент, кнопка «сохранить как картинку» (1080×1080 PNG для соцсетей).
- **`bot.py`** — Telegram-бот на `python-telegram-bot`. Команды `/wish`, `/daily`, `/myid` + ежедневная авторассылка пожелания + случайного комплимента.

## 🚀 Запуск

### Сайт
Просто открой `index.html` в браузере — всё работает без сервера.

### Бот
```bash
pip install -r requirements.txt
cp .env.example .env  # и заполни BOT_TOKEN, OWNER_ID
python bot.py
```

Переменные окружения (см. `.env.example`):
- `BOT_TOKEN` — токен от [@BotFather](https://t.me/BotFather)
- `OWNER_ID` — твой Telegram id (для ежедневной рассылки)
- `TZ`, `DAILY_HOUR`, `DAILY_MIN` — расписание (по умолчанию 10:00 МСК)
